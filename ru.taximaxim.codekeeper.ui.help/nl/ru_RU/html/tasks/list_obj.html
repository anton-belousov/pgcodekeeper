Сохранение наборов объектов

Набор представляет из себя список объектов, которые должны быть выбраны и отмечены флажком.
Наборы удобно использовать для сохранения выбора объектов между сравнениями разных БД, разных проектов. Набор так же возможно экспортировать в виде регулярного выражения для передачи другому человеку для использования в качестве фильтра объектов.

Сохранение набора:

  • Выбрать объекты, которые должны быть сохранены в набор
  • Ввести имя нового набора в выпадающий список
  • Нажать кнопку Сохранить набор выбранных элементов
  • В выпадающем списке Сохраненные наборы появиться элемент с именем сохраненного списка

Удаление набора:

  • В выпадающем списке Сохраненные наборы выбрать набор который необходимо удалить
  • Нажать кнопку Удалить набор выбранных элементов

Экспорт набора в буфер обмена:

  • В выпадающем списке Сохраненные наборы выбрать набор который необходимо экспортировать в буфер или вручную выбрать необходимые элементы
  • Нажать кнопку Копировать набор выбранных элементов в буфер обмена. Список объектов будет скопирован в буфер обмена в виде регулярного выражений. Эту строку можно использовать в поле поиска, и в панели различий отобразятся только те объекты, которые содержатся в наборе.
