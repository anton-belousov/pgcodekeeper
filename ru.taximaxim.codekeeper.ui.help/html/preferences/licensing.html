Лицензия

Лицензия pgCodeKeeper накладывает ограничения на период использования и количество объектов сравниваемых баз данных. По умолчанию лицензия действует 2 месяца после выхода последнего релиза и ограничена 100 объектами базы данных. Объектами базы данных в данном случае считаются: domain, function, table, view, type.

Присутствует возможность получить лицензию, в которой будут полностью или частично сняты ограничения. Для этого необходимо отправить запрос по адресу codekeeper@chelny.taximaxim.ru с указанием электронного адреса, полного имени, и ограничений по лицензии: предельное количество объектов, срок действия лицензии, графическая или консольная. В случае положительного решения по вашей заявки на ваш электронный адрес будет выслан файл лицензии. Для активации Вам потребуется сохранить файл лицензии на диск и на странице настроек pgCodeKeeper -> Лицензия в поле Файл лицензии указать данный файл. Путь к имени файла должен быть абсолютный.

После добавления файла лицензии измениться информация о лицензии и ограничениях.
