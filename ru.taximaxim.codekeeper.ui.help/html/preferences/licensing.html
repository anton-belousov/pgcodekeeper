Лицензирование

Лицензия pgCodeKeeper накладывает ограничения на период использования продукта и количество объектов, содержащихся в схемах сравниваемых баз данных. В продукт встроена пробная лицензия, действующая в течение двух месяцев после выпуска и ограниченная 100 объектами в схеме базы данных.

  • Подсчет объектов схемы базы данных в контексте лицензирования происходит по следующим классам: domain, type, function, table, view.

Запросы на получение лицензии с полностью или частично снятыми ограничениями принимаются по адресу pgcodekeeper@chelny.taximaxim.ru с указанием электронного адреса, имени либо наименования получателя, и желаемых характеристик лицензии: количества обрабатываемых объектов, срока действия лицензии, типа лицензируемого продукта (графический или консольный).
В случае положительного решения по вашей заявке на ваш электронный адрес будет выслан файл лицензии. Для активации вам потребуется сохранить файл лицензии на диск и на странице настроек pgCodeKeeper -> Лицензия в поле Файл лицензии указать данный файл.
На данный момент условия лицензирования рассматриваются индивидуально. По любым вопросам, связанным с лицензированием пишите на pgcodekeeper@chelny.taximaxim.ru.

После добавления файла лицензии изменится отображаемая информация о лицензии и ограничениях.
