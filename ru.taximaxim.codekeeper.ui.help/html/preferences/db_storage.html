Хранилище БД

Настройки, управляющие хранилищем параметров подключения к БД.

На странице настроек pgCodeKeeper -> Хранилище БД находится список записей данных подключения к базам данных.

Для удаления хранилища выбрать необходимую запись и нажать кнопку удалить

Для добавления хранилища нажать кнопку добавить . Откроется диалог Данные подключения к БД в котором нужно указать параметры данных подключения: название записи, имя БД, пользователь, пароль, хост, порт. Нажать на кнопку OK. На странице настроек нажать кнопку Apply или OK и хранилище БД будет сохранено.

Для редактирования хранилища выделить объект хранилища баз данных, нажать кнопку редактировать . Откроется диалог Данные подключения к БД в котором можно изменить параметры подключения. Нажать на кнопку OK. На странице настроек нажать кнопку Apply или OK и список хранилищ будет сохранен.
