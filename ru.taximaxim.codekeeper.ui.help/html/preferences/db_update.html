Обновление Базы Данных

Настройки управляющие обновлением базы данных.

Показать вывод скрипта наката в отдельном окне, отобразит результат выполения скрипта в отдельном окне.

Разрешить PostgreSQL проверять тела функций, при установке галочки уберет из начала скрипта наката выражение SET check_function_bodies = false;

Можно удалять и добавлять команды скрипта наката в нижней части.
