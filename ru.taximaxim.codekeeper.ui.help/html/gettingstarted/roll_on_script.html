Редактор скрипта наката изменений

Редактор скрипта наката изменений представляет из себя панель настройки и SQL редактор pgCodeKeeper.

Панель настройки предоставляет возможность для выбора и способа обновления базы данных. Обновить DDL можно как через JDBC, так и через любую удобную для Вас утилиту. По умолчанию представлена утилита psql и шаблон для выполнения обновления.

В поле SQL редактора pgCodeKeeper представлен сгенерированный скипт наката, приводящий состояние целевого объекта внешней схемы бд к состоянию исходного объекта. Данная область имеет такие же возможности что и обычный SQL редактора pgCodeKeeper

В этом окне представлен скрипт наката, позволяющий привести состояние объекта из одной БД в состояние в другой.
