Основная панель проекта

Основная панель проекта состоит из трех основных частей:

  • 1. Панель сравнения
  • 2. панель выбора источника БД
  • 3. Панель изменений

Панель сравнения отображает список объектов, которые имеют отличия в сравниваемых схемах БД. Изменение могут быть 3-х видов: удаление, создание, редактирование. В зависимости от типа изменения объкты в списке изменений помечаются как delete, new и edit соответственно.

В нижней части панели сравнения расположена панель инструментов предназначенная для работы со списком объектов. Они имеют следующее предназначение:

  • - сбросить сортировку
  • - выделить все элементы
  • - снять выделение со всех элементов
  • - инвертировать выделение элементов
  • - сохранить набор выбранных элементов
  • - удаление набора выбранных элементов
  • - список сохраненных наборов выделенных элементов
  • - копирование набора выбранных элементов в буфер обмена

В верхней части панели распологается поле поиска объекта по имени

Панель выбора источника БД содержит параметры подключения к схеме БД, с которой будет сравниваться схема проекта. Имеется возможность ввода параметров в ручном так и в автоматичском режиме при помощи, настройки хранилищ базы данных.

Панель изменений показывает изменения в SQL запросе объекта
