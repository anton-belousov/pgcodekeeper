Установка плагина pgCodeKeeper в Eclipse

Самый простой путь получить pgCodeKeeper через обновление программного обеспечения: из запущенного экземпляра Eclipse 4.2 или выше, выбрать Help > Install New Software.... Появится мастер установки плагинов средствами update site.

В поле Work With нажать на кнопку Add и в диалоговом окне в поле Location ввести путь к репозиторию. Это может быть как удаленный ресурс http://pgcodekeeper.ru/update/release/ так и локальный в вашей файловой системе. В поле Name вводиться удобное имя, например pgCodeKeeper.

Выбираем пакет pgCodeKeeper, нажимаем Next, Next, соглашаемся с пользовательским соглашением, нажимаем кнопку Finish и начинается установка.

По завершению установки для применения изменений необходимо презагрузить Eclipse

Запустите Eclipse Workbench и выберите Window > Perspective > Open Perspective > Other > pgCodeKeeper.
