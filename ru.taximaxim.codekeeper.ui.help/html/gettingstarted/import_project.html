Импортирование существующего проекта

Импорт существующего проекта происходит через мастер создания нового проекта.

Файл проекта называется ".project", также проект содержит директорию ".settings", в которой находятся настройки проекта.

Если файл проекта существует, то импорт производится путем импортирования существующего проекта в Eclipse:

  • 1. Из меню выберите пункт File > Import... Откроется выбор мастеров импорта.
  • 2. Выберите категорию General > Existing Project into Workspace и нажмите Next.
  • 3. Выберите директорию содержащую файл проекта.
  • 4. Затем выберите проекты для импорта.
  • 5. Закончите импорт нажатием на Finish.

Если файла проекта не существует, например импортируется проект из директории GIT репозитория, не содержащего файла проекта, то выполните следующее:

  • 1. Из меню выберите пункт File > Project... Откроется выбор мастеров новых проектов.
  • 2. Выберите категорию pgCodeKeeper выберите pgCodeKeeper Project и нажмите Next.
  • 3. Введите имя проекта и укажите путь до директории Git репозитория. Нажмите Next.
  • 4. Нажмите Next еще раз.
  • 5. Нажмите Finish.
