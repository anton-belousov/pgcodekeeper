Импорт проекта

Если имеется проект Eclipse (такой проект имеет файл .project в корне), то производится импорт существующего проекта:

  1. Из меню выберите пункт File > Import... Откроется выбор мастеров импорта.
  2. Выберите категорию General > Existing Project into Workspace и нажмите Next.
  3. Выберите директорию проекта.
  4. Затем выберите проекты для импорта.
  5. Закончите импорт нажатием на Finish.

В случае, если присутствует только структура проекта pgCodeKeeper, без файлов проекта Eclipse, то импорт производится с помощью мастера нового проекта.
Хранение структуры проекта pgCodeKeeper без информации Eclipse – обычная практика, например при использовании системы контроля версий.

  1. Из меню выберите пункт File > New > Project... Откроется выбор мастеров новых проектов.
  2. Выберите pgCodeKeeper Project в категории pgCodeKeeper и нажмите Next.
  3. Введите имя проекта, снимите флажок Use default location и укажите путь до директории со структурой проекта pgCodeKeeper.
  4. Нажмите Finish.
  5. Импорт через мастер нового проекта возможен только при наличии файла ".pgcodekeeper" в корне импортируемой директории. Иначе запустится процесс создания нового проекта в указанной директории с нуля.
