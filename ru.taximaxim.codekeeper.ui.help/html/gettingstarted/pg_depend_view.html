Просмотр зависимостей объекта бд

pg Зависимости view показывает отношение между объектами бд относительно выбранного элемента из списка отличий, полученных при сравнении двух баз данных. Кнопки Source и Target являются переключателем для отображения связей исходной и целевой базы данных. После их переключения необходимо повторно щелкнуть на элемент в списке изменений.

Стрелками указаны направления зависимости от зависящего объекта к зависимости
