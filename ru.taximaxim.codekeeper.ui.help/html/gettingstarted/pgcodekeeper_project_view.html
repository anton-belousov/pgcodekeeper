Редактор проекта pgCodeKeeper

Редактор проекта – основной способ работы с pgCodeKeeper.
Две вкладки, расположенные у нижнего края редактора, позволяют переключаться между режимом внесения изменений в проект и генерации скрипта наката изменений на БД.
Вкладки имеют схожий интерфейс, состоящий из трех основных частей:

  1. Панель задач
  2. Панель инструментов
  3. Таблица изменений
  4. Панель изменений

Панель задач содержит набор действий для сравнения и обновления баз данных:

  • Получить изменения
  • Сгенерировать скрипт
  • Добавить зависимости
  • Применить выбранные изменения...

    • Панель инструментов предназначена для работы со списком объектов. Они имеют следующее предназначение:

    • – поле фильтра списка объектов по имени или регулярному выражению.
    • – сбросить сортировку
    • – выбрать все элементы
    • – снять выбор со всех элементов
    • – инвертировать выбор элементов
    • Выпадающий список с сохраненными наборами объектов. Позволяет восстановить выбор объектов согласно сохраненному набору.
    • – сохранить набор выбранных элементов
    • – удалить текущий сохраненный набор выбранных элементов

    Таблица изменения отображает список объектов, которые различаются в сравниваемых схемах БД. Различие может быть одного из трех видов: удаление, создание или изменение объекта. В зависимости от типа различия, объекты в списке изменений помечаются как delete, new и edit соответственно.

    Панель изменений показывает изменения, произошедшие в SQL представлении объекта.
