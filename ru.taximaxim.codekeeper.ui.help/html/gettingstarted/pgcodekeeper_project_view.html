Редактор проекта pgCodeKeeper

Редактор проекта – основной способ работы с pgCodeKeeper.
Две вкладки, расположенные у нижнего края редактора, позволяют переключаться между режимами вноса изменений в проект и генерации скрипта наката изменений на БД.
Вкладки имеют схожий интерфейс, состоящий из трех основных частей:

  1. Панель различий
  2. Панель выбора используемой БД
  3. Панель сравнения

Панель различий отображает список объектов, которые различаются в сравниваемых схемах БД. Различие может быть одного из трех видов: удаление, создание или изменение объекта. В зависимости от типа различия, объекты в списке изменений помечаются как delete, new и edit соответственно.

У верхнего края панели различий располагается поле фильтра списка объектов по имени или регулярному выражению.

У нижнего края расположен набор инструментов для работы со списком объектов. Они имеют следующее предназначение:

  • – сбросить сортировку
  • – выбрать все элементы
  • – снять выбор со всех элементов
  • – инвертировать выбор элементов
  • Выпадающий список с сохраненными наборами объектов. Позволяет восстановить выбор объектов согласно сохраненному набору.
  • – сохранить набор выбранных элементов
  • – удалить текущий сохраненный набор выбранных элементов

Панель выбора используемой БД содержит параметры подключения к БД, со схемой которой будет сравниваться проект. Имеется возможность заполнения параметров как вручную, так и автоматически, из хранилища баз данных.

Панель сравнения показывает изменения, произошедшие в SQL представлении объекта.
