Создание проекта pgCodeKeeper

Открываем мастер создания нового проекта: File -> New -> Project...

Выбираем Проект pgCodeKeeper. Нажимаем кнопку Next.

Указываем имя проекта. Нажимаем кнопку Next.

Выбираем источник схемы для инициализации проекта: подключение JDBC, снятие дампа pg_dump или сторонний файл дампа.
Здесь же вы можете сохранить настройки подключения к БД с помощью хранилища баз данных. Для этого, выбрав JDBC или pg_dump, нажмите на иконку редактирования хранилища БД справа от выбора источника БД. Появится окно редактирования хранилища БД.
Для файла дампа достаточно указать путь к файлу вручную или нажав на кнопку Обзор... и выбрав файл в появившемся диалоговом окне.

После выбора источника схемы нажимаем кнопку Finish.

После короткого ожидания получаем новый, инициализированный схемой указанной БД, и автоматически открывается Редактор проекта pgCodeKeeper.
