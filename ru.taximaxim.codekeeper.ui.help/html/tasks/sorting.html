Сортировка списка изменений

Для сортировки объектов в списке измений необходимо кликнуть по заголовку столбца.

По умолчанию используется множественная сортировка. При сортировке по новому стобцу учитывается сортировка по предыдущим выбранным столбцам.

Для сброса сортировки можно нажать на кнопку сбросить сортировку на панели инструментов или кликнуть на заголовок столбца удерживая клавишу Ctrl.

Числа и стрелки в заголовках таблицы показывают порядок и направление сортировки.
