Поиск объекта базы данных

Для поиска объектов в панели сравнения удобно воспользоваться полем поиска над списком изменений.

Поиск происходит по имени объекта двумя способами:

  • используя в качестве параметра часть имени объекта.
  • используя в качестве параметра регулярное выражение. В этом случае необходимо включить опцию установив флажок в поле RegEx

Для начала поиска необходимо просто ввести часть имени в поле поиска. В панели сравнения отобразиться список объектов, подходях данному условию.
