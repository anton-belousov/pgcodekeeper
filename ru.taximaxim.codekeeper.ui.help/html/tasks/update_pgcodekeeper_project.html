Обновление pgCodeKeeper проекта

На основной панели проекта переключаемся в режим обновления проекта (в нижней части панели переключаемся на вкладку Обновить проект).

На панели выбора источника БД указываем параметры для получения схемы БД на основании которой будет обновляться pgCodeKeeper проект.

На панели действий нажать кнопку Получить изменения. После недолгого ожидания на панели сравнения отображается список различий с указанием типа изменения.

Флажками отметить объекты которые мы хотим обновить. Нажать на кнопку Применить выбранные изменения.... Откроется диалог обновления проекта. Проверить список объектов выбранных пользователем и список зависимых объектов. Будьте внимательны! Добавление и удаление зависимостей может привести к ошибкам или неожиданным результатам при обновлении проекта.

Для подтверждения применения измененений к проекту нажать кнопку OK. В случае успешного применения изменений в Console view появиться соответствующее сообщение.
